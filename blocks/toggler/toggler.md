## Yate

### Default toggler

```
nb-toggler()
```

* leftText — On
* rightText — Off
* size — s

### Optional settings
* `'id'` {string}
* `'name'` {string}
* `'disabled'` {boolean}
* `'checked'` {boolean}
* `'leftText'` {string} 'On'
* `'rightText'` {string} 'Off'

### Example

Turned on enabled toggler with custom a titles on the sides

```
nb-toggler({
    'leftText': 'yep!'
    'rightText': 'nope :-('
    'checked': true()
})
```
